# VoiceSample IG v0.1

This Implementation Guide defines **VoiceSample**, a minimal profile on FHIR R4 `Media` representing a **single clinical voice capture** with analysis-relevant metadata and standard linkages to derived `Observation` resources and `Provenance`.
