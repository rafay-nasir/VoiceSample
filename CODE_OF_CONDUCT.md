# Code of Conduct

This project adopts the **Contributor Covenant** (version 2.1) as its code of conduct.

The full text is published at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In brief: contributors and maintainers commit to making participation in this project a respectful, welcoming, and professional experience for everyone, and to handling reports of inappropriate behavior fairly and consistently.

## Reporting

Until the project formalizes a steering group, concerns about conduct in this repository should be raised by opening a GitHub issue marked `conduct` or by contacting the project lead through GitHub.

## Scope

This Code of Conduct applies within all project spaces (issues, pull requests, discussions, related communications) and when an individual is representing the project in public spaces.
